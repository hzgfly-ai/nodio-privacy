<h1>Nodio 隐私政策</h1>
<p class="date">最后更新日期：2026年2月25日</p>

<p>Nodio（以下简称"本应用"）尊重并保护用户的隐私。本政策说明我们如何收集、使用和保护您的信息。</p>

<h2>1. 笔记数据存储</h2>
<p>您的笔记数据主要存储在您的设备本地。如您已登录 iCloud，笔记数据会通过 Apple CloudKit 私有数据库同步至您的 iCloud 账户，仅您本人可访问，我们无法读取。</p>

<h2>2. AI 服务</h2>
<p>本应用使用<strong>阿里云通义千问</strong> AI 服务进行文本分析、自动分类、排版和润色。当您保存笔记或使用 AI 功能时，笔记文本内容会发送至该服务处理，我们不会在服务器端存储您的笔记内容。</p>
<p>阿里云隐私政策：<a href="https://terms.aliyun.com/legal-agreement/terms/privacy" target="_blank">https://terms.aliyun.com/legal-agreement/terms/privacy</a></p>

<h2>3. 语音识别</h2>
<p>语音输入功能使用两种引擎：</p>
<ul>
  <li><strong>腾讯云实时语音识别（联网时）</strong>：语音音频流发送至腾讯云服务器识别，识别后不留存。隐私政策：<a href="https://cloud.tencent.com/document/product/301/11470" target="_blank">腾讯云隐私声明</a></li>
  <li><strong>Apple 原生离线语音识别（无网络时）</strong>：语音数据在本地处理，不经过任何服务器。</li>
</ul>

<h2>4. 书籍封面搜索</h2>
<p>当您记录书籍摘抄时，本应用会向 Google Books、Open Library、iTunes Search API 查询书籍封面图片，仅发送书名信息，不包含任何用户身份数据。</p>

<h2>5. 匿名使用统计</h2>
<p>本应用会收集以下匿名统计数据，用于改善产品体验：</p>
<ul>
  <li>随机生成的匿名设备标识符（不与您的身份关联）</li>
  <li>应用版本、系统版本</li>
  <li>每日笔记创建数量、AI 使用次数等聚合数字</li>
  <li>应用活跃时长</li>
</ul>
<p>上述数据通过 Apple CloudKit 公开数据库上报，<strong>不包含任何笔记内容或个人身份信息</strong>。</p>

<h2>6. 权限说明</h2>
<ul>
  <li><strong>麦克风</strong>：仅在您主动点击录音时激活，不会在后台录音。</li>
  <li><strong>语音识别</strong>：用于将语音转换为文字，作为离线识别备用引擎。</li>
  <li><strong>推送通知</strong>：仅用于接收 iCloud 数据变更的静默推送，实现多设备实时同步，不会发送营销通知。</li>
</ul>

<h2>7. 我们不会收集的信息</h2>
<ul>
  <li>不收集用户身份信息（姓名、邮箱、手机号等）</li>
  <li>不使用任何第三方广告或分析 SDK</li>
  <li>不将笔记内容用于任何商业目的</li>
  <li>不与任何第三方共享用户数据</li>
</ul>
