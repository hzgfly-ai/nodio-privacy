<h1>Nodio 隐私政策</h1>
<p>最后更新日期：2026年2月25日</p>

<h2>1. 数据存储</h2>
<p>本应用的所有笔记数据均存储在您的设备本地，我们不会将您的笔记内容上传至我们的服务器。</p>

<h2>2. AI 服务</h2>
<p>本应用使用第三方 AI 服务（Moonshot AI / Kimi）进行文本分析、分类和排版。当您保存笔记或使用 AI 排版功能时，笔记文本内容会发送至该服务进行处理。我们不会存储或缓存 AI 处理的中间数据。</p>
<p>第三方服务隐私政策：<a href="https://www.moonshot.cn/privacy">https://www.moonshot.cn/privacy</a></p>

<h2>3. 网络请求</h2>
<p>本应用仅在以下场景发起网络请求：</p>
<ul>
  <li>AI 文本分析与排版（Moonshot AI API）</li>
  <li>书籍封面搜索（Open Library、iTunes Search API、Google Books API）</li>
</ul>
<p>上述请求不包含任何用户身份信息。</p>

<h2>4. 语音数据</h2>
<p>本应用使用 Apple 内置的语音识别服务，语音数据由 Apple 处理，不经过我们的服务器。</p>

<h2>5. 麦克风权限</h2>
<p>本应用需要麦克风权限以提供语音输入功能，仅在您主动使用语音输入时激活，不会在后台录音。</p>

<h2>6. 我们不会收集的信息</h2>
<ul>
  <li>不收集用户身份信息（姓名、邮箱、手机号等）</li>
  <li>不收集设备标识符用于追踪</li>
  <li>不使用任何第三方分析或广告 SDK</li>
  <li>不与任何第三方共享用户数据</li>
</ul>
